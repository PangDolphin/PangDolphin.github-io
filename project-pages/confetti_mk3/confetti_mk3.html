Confetti Dispenser Mk 3

Programming language: Python, C++

This project was for my final year 12 project and was featured in the 2024 Pycon student showcase. This was a improvement to the Rapid Firing Confetti Dispenser. Featuring, autonomous infrared aiming (powered by computer vision), GUI, remote control aiming, and a belt or magazine feeding mechanism (interchangeable).
